Zomato Dataset Analysis

Project Overview

This project analyzes the Zomato restaurant dataset using Python to identify trends in restaurant ratings, popular cuisines, restaurant locations, and pricing patterns. The analysis provides insights into customer preferences and factors influencing restaurant performance.

Objectives

- Analyze restaurant ratings and customer satisfaction.
- Identify the most popular cuisines.
- Determine restaurant distribution across different locations.
- Examine the relationship between restaurant cost and ratings.
- Generate insights using data visualization and exploratory data analysis.

Dataset Description

The dataset contains information about restaurants, including:

- Restaurant Name
- Location
- Cuisines
- Rating
- Votes
- Average Cost for Two
- Online Order Availability
- Table Booking Availability
- Restaurant Type

Key Findings

- Most restaurants have ratings between 3.5 and 4.5.
- North Indian and Chinese cuisines are among the most popular.
- Certain locations have a higher concentration of restaurants.
- Restaurant cost has a weak correlation with ratings.
- Restaurants with higher vote counts generally receive better ratings.
